# Coby's Charizard

Pokemon side-scrolling shooter for Coby (Colin's 6yo nephew). Shares engine with Grayson's Lugia.

## Links
- **Live:** https://colin1201.github.io/cobys-charizard/
- **Repo:** https://github.com/colin1201/cobys-charizard
- **Shared engine:** `../pokemon-game/`

## What's different from Grayson's version
- Rides Charizard instead of Lugia
- Flamethrower attack (orange/red) instead of Aeroblast (blue)
- "Coby's Charizard" title
- Separate high score storage

## How to update
Edit `../pokemon-game/engine.html`, run `node ../pokemon-game/build.js`, then push this repo.
